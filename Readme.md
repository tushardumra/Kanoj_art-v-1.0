📌 Overview

Kanoj Art is a modern and visually appealing digital art showcase website designed to present creative artwork in an elegant and user-friendly interface. The platform highlights different art pieces with smooth navigation, responsive design, and engaging UI elements.

This project focuses on delivering a clean aesthetic experience while ensuring performance and accessibility across devices.

✨ Features
🎨 Beautiful and modern UI design
📱 Fully responsive (mobile, tablet, desktop)
🖼️ Art gallery showcase section
🚀 Fast loading and optimized performance
🧭 Smooth navigation and user experience
📩 Contact / interaction section (if implemented)
🌙 Clean typography and layout

🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
Styling: CSS (Flexbox/Grid)
Backend: Formspree (Baas)
Deployment: Vercel
